# HTR Study

HTR stands for Half Turn Reduction. It is an advanced method for solving the Rubik's cube with fewer moves than traditional methods, in a human findable way. The HTR method focuses on reducing the number of half turns (180-degree turns) required to solve the cube, which can lead to more efficient solutions.

For now, this is not a guide. This is personal study notes shared with people who already understand the basics of the HTR method.

# Requirements

- Python 3.10 or higher
- Virtual environment (optional but recommended)

## Get Started

This assumes you are in the `htr` directory and have Python.

```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
manim -ql htr.py HTR
```
